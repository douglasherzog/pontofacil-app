# Instalação (ambiente de desenvolvimento)

## Requisitos

- Node.js (para o monorepo com pnpm)
- pnpm (ver versão no `package.json` do root, campo `packageManager`)
- Python 3 (para a API)
- Pandoc (para geração do PDF do manual)

## Pandoc no Windows

Instale o Pandoc e garanta que o comando `pandoc` funcione no terminal.

Verificação:

- `pandoc --version`
